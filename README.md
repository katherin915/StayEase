# 🏨 StayEase – Hotel Booking Frontend Project

## 📌 Overview
StayEase is a frontend-based hotel booking web application designed to provide users with a simple and user-friendly platform to explore hotels, check availability, and book rooms.

This project focuses on clean UI, smooth navigation, and interactive features using HTML, CSS, and JavaScript.

---

## 🚀 Features

- 🔍 **Search Hotels** – Real-time filtering of hotels based on user input  
- 📅 **Date Picker** – Integrated Flatpickr for selecting check-in and check-out dates  
- 🏨 **Hotel Listings** – View available hotels with details and pricing  
- 📝 **Booking Form** – Users can enter details to book rooms  
- 💰 **Dynamic Price Calculation** – Total price updates based on number of rooms  
- ✅ **Form Validation** – Ensures correct user input before submission  
- 📱 **Responsive Design** – Works across different screen sizes  

---

## 🛠️ Tech Stack

- **HTML5** – Structure  
- **CSS3** – Styling & Responsive Design  
- **JavaScript** – Interactivity & DOM Manipulation  
- **Flatpickr** – Date & Time Picker Library  
- **Font Awesome / Remix Icons** – Icons

## 🔄 How It Works

1. User lands on homepage  
2. Selects location, date, and guests  
3. Navigates to availability page  
4. Filters/searches hotels  
5. Selects a hotel  
6. Fills booking form  
7. Views dynamic pricing and submits details
